# WatchLLM Roadmap

Master roadmap and product milestones for the WatchLLM ecosystem.

## Milestones

1. **Phase 1 — Kernel MVP:** Build deterministic local write-path engine (Complete).
2. **Phase 2 — Wasm Runtime:** Rust-port and compilation for embeddable zero-process latency (In Progress).
3. **Phase 3 — Editor Integration:** VS Code save hook extension (Scaffolded).
4. **Phase 4 — klyd Integration:** Architectural memory sync.
5. **Phase 5 — Replay Engine:** Observability trace timeline.
